Contact / Submit Achievement

Use this form to submit a student's achievement. Teachers or parents may submit once authenticated by school email (future feature).

Pseudocode (Form Validation)

BEGIN
  READ name, category, email, message
  IF any field is empty THEN
     SHOW "Please complete all fields"
     STOP
  END IF
  IF email is not valid THEN
     SHOW "Enter a valid email"
     STOP
  END IF
  SEND data to server API endpoint /api/submitAchievement
  SHOW "Thank you — submission received"
END
